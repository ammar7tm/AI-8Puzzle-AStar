# AI Final Project – 8 Puzzle Solver using A*

This project implements the A* search algorithm to solve the 8-puzzle problem.

The 8-puzzle consists of a 3×3 board with eight numbered tiles and one empty space. 
The objective is to rearrange the tiles until they reach the correct goal configuration.

The project will use heuristic functions such as Manhattan Distance and Misplaced Tiles 
to guide the search and compare their performance.
